# **App Name**: Soft Echoes

## Core Features:

- Welcome Message: Display a gentle, welcoming message with a subtle fade-in animation to greet the visitor.
- Scrolling Timeline: Showcase key moments with aesthetic card components that smoothly slide in as the user scrolls through the timeline.
- Memory Gallery: Present cherished photos in a Pinterest-style grid with soft borders, rounded corners, and pastel shadows for an airy feel.
- Emotional Letter: Create a heartfelt letter section with a handwritten-style font and a soft, paper-like background for an intimate touch.
- Final Message: Convey a reassuring closing message with a gentle, floating heart or sparkle animation, subtly reinforcing the message of enduring affection.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke calm and affection.
- Background color: Creamy pastel (#F8F8FF), a barely-there hue close to white, providing a gentle backdrop that doesn't distract from the content.
- Accent color: Baby blue (#ADD8E6) for interactive elements and highlights.
- Headline font: 'Dancing Script' (serif) for headings, emphasizing the handwritten, emotional vibe. Note: currently only Google Fonts are supported.
- Body font: 'Poppins' (sans-serif) for the body, keeping readability high. Note: currently only Google Fonts are supported.
- Ensure generous spacing and padding to maintain a clean and airy aesthetic.
- Use subtle fade and slide-in animations to add a touch of gentle movement.
